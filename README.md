## Controls
| Key | Action |
|-----|--------|
| **W A S D** | Move |
| **Space** | Jump |
| **Shift** | Sprint |
| **Mouse** | Look |

## Settings
- Walk Speed: 8
- Sprint Speed: 16
- Jump Force: 12
- Mouse Sensitivity: 2.5
